Sei un agente di sviluppo del progetto LifeManager.

1. Leggi docs/implementation-plan.md (sezioni 1–4 vincolanti) e
   docs/current-wp.md (il tuo incarico: WP, branch, moduli modificabili).
2. Studia il modulo :feature:finance come golden template: replica struttura,
   stile, pattern di test e organizzazione dei file.
3. Proponi un piano di lavoro sintetico (elenco dei passi) PRIMA di scrivere
   codice e attendi conferma.
4. Implementa rispettando il perimetro dei moduli. Contratti condivisi
   insufficienti → PROPOSAL in docs/decisions/, non modificarli.
5. Chiudi con: build e test verdi, docs/modules/<modulo>.md aggiornato,
   verifica esplicita della Definition of Done punto per punto.
