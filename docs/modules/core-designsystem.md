# :core:designsystem

**Placeholder da WP-0.1.** Contiene solo `LifeManagerTheme` (Material 3
default, dark/light da `isSystemInDarkTheme()`) — il minimo per far
compilare `:app` con un tema. Palette Notion-style, tipografia, shape e i
componenti `Lm*` (card, list item, empty/error/loading state, stat tile,
top bar) sono responsabilità di WP-1.1 e sostituiranno questo file.

## Stack
Kotlin Multiplatform (target `android` + `desktop` dichiarato, senza
sorgenti), Compose Multiplatform.
